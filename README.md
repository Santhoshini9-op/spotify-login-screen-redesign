# Spotify Login Screen Redesign

## Overview

This project is a recreation of Spotify's login screen designed in Figma. The goal was to practice UI design fundamentals by analyzing a real-world product and rebuilding the interface while maintaining usability and visual consistency.

## Tools Used

* Figma

## Skills Practiced

* Typography
* Color Hierarchy
* Visual Hierarchy
* Spacing and Alignment
* Auto Layout
* Components
* Mobile UI Design
